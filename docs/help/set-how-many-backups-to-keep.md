category: Settings & data

Backups add up over time. Retention sets how many to keep for each app — older
ones beyond that count are trimmed so your list and storage stay tidy. You can
set a default for all apps and override it per app.

## Steps

1. To set the default for every app, open **Settings**, find **Default
   retention**, and enter the number of backups to keep per app.
2. To override one app, open the **Apps** tab, find its card, and change the
   **Retention** number there.
3. To trim older backups right now, open **Settings** and tap **Prune now**
   under **Default retention**.

## What you'll see

The numbers save as you change them. **Prune now** reports how many backups were
removed (or that there was nothing to trim). Pruning removes older backups
beyond each app's limit from the list.
