Restoring takes the data from a saved backup and loads it back into an app —
for recovering after a data wipe, undoing a bad change, or moving data to a new
device. You restore one app at a time so you stay in control of what's replaced.

## Before you start

- Restoring **replaces** all current data in that app with the snapshot. Make a
  fresh backup first if the current data still matters.

## Steps

1. Open the **History** tab from the bottom bar.
2. Find the backup you want and tap **Open**.
3. Under **Apps in this bundle**, find the app you want to restore.
4. Tap **Restore** next to it.
5. Read the confirmation — it names the app and the snapshot's date — then
   confirm with **Restore**.

## What you'll see

A confirmation appears when the restore finishes. Open that app to see its data
returned to the state captured in the backup.
