Backups are written to durable storage outside the browser so they survive a
data wipe. Where they go depends on your device — set this up once and backups
become quick and predictable.

## Steps

1. Open the **Settings** tab from the bottom bar.
2. Find the **Backup folder** card.
3. On a supported browser (Android, desktop Chrome or Edge), tap **Choose
   folder** and pick the folder where bundles should be written. Use **Change
   folder** later to switch.
4. On iPhone or iPad, the card explains that backups use the system share sheet
   instead. There's nothing to choose ahead of time — each backup opens the Save
   sheet so you can pick **Files** or **iCloud Drive**.

## What you'll see

When a folder is set, Home shows a **Folder** badge with its name, and backups
are written there silently. Without a folder (or on iPhone and iPad), each
backup opens the share or download sheet so you can save the file yourself.
