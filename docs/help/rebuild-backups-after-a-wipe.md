summary: Re-index your saved backup files on a fresh browser or new device.
---

The list of backups lives in the browser, so clearing site data or moving to a
new device empties it — even though your saved backup files are safe on disk or
in iCloud. Re-importing those files rebuilds the list so you can restore from
them again.

## Steps

1. Open the **Settings** tab from the bottom bar.
2. Find the **Import backup files** card.
3. Use the file picker to select the previously-saved `.json` backup files —
   you can pick several at once.
4. If you back up to a chosen folder (Android or desktop), you can instead open
   the **Backup folder** card and tap **Rescan folder** to pull in everything
   already saved there.

## What you'll see

A summary reports how many backups were newly imported, how many were already
known, and how many failed. The imported backups reappear under
[History](/help/browse-your-backup-history), ready to restore.
