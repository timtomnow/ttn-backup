ttn-backup keeps your other apps safe. The apps you use all store their data
inside the browser, so a single "clear site data" can wipe everything at once.
ttn-backup gathers every app's data into one file and saves it somewhere
durable, so you can put it back any time.

## Before you start

- Decide where backups should be saved. On Android you can pick a folder; on
  iPhone and iPad each backup goes through the system share sheet so you can
  save it to Files or iCloud Drive. See [Choose where backups are saved](/help/choose-where-backups-are-saved).

## Steps

1. Open the **Home** tab from the bottom bar.
2. Check the **Per-app status** list — each app shows when it was last backed
   up, or **No backups yet**.
3. Tap **Backup all apps now**.
4. If a save sheet appears, choose where to save the file and confirm.

## What you'll see

A confirmation appears and the **Bundles stored** and **Last backup** numbers at
the top of Home update. Your data is now captured in a single backup file. From
here you can set up automatic reminders or restore an app whenever you need to.
