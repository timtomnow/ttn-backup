A full backup snapshots every app at once and saves them together in a single
file. This is the backup you'll run most often.

## Before you start

- Decide where the file should go — see [Choose where backups are saved](/help/choose-where-backups-are-saved).

## Steps

1. Open the **Home** tab from the bottom bar.
2. Tap **Backup all apps now**. (You can also use **Backup all** in the top
   bar from any screen.)
3. If a save sheet appears, choose where to save the file and confirm.

## What you'll see

A confirmation appears noting how the bundle was saved, and the **Bundles
stored** and **Last backup** figures update. If any app couldn't be reached, a
summary lists which ones were skipped — the rest are still saved. The new bundle
shows up under [History](/help/browse-your-backup-history).
