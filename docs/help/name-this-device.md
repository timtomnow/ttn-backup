If you back up from more than one device, a device label helps you tell which
backup came from where. The label is saved inside each backup file you make from
this device.

## Steps

1. Open the **Settings** tab from the bottom bar.
2. Find the **Device label** card.
3. Type a name, for example *iPhone 15*.
4. Tap or click away from the field to save it.

## What you'll see

The label is stored and included in every backup made from this device going
forward. It's optional — leave it blank if you only ever back up from one place.
